# WGET

This project objective consists on recreating some functionalities of [wget](https://www.gnu.org/software/wget/manual/wget.html).

## Run the project

To build the project:

```

go build wget

```

And then to run:

```

./wget [Your args]

```

To run manually:

```

go run . [Your args]

```
